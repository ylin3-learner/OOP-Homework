Car Service Management System

## Overview
This project is a Car Service Management System that allows users to:

- Add customers and cars

- Create service quotes

- View and delete service quotes

#### It provides an interactive menu-driven interface to manage customer information, vehicle details, and service costs.

Features

- Customer Management: Add and display customer details

- Car Management: Add and display car information

- Service Quotes: Generate and manage service quotes

- Data Persistence: Uses data structures to store customer, car, and quote information
